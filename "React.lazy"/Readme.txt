⚠️ Things to Know

| Rule                              | Description                        |
| --------------------------------- | ---------------------------------- |
| Must use inside `<Suspense>`      | To show fallback while loading     |
| Only default exports work         | Lazy only supports default exports |
| Can be combined with `React.memo` | For even more optimization         |
| Doesn't support named exports     | Wrap in another file if needed     |
